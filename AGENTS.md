# AGENTS.md

## Rules

- Always use `pnpm` as the package manager (never npm or yarn).
- Follow KISS (Keep It Simple, Stupid) principle — prefer the simplest solution that works.
- Follow DRY (Don't Repeat Yourself) principle — avoid code duplication.
- Use best practices for all solutions.
- Search for solutions on the internet when facing hard or unfamiliar problems.
- Do not run build commands; only check for lint errors (`pnpm lint`).
